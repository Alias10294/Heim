# Heim: A Simple But Fast ECS

Meet `Heim`, my very first **Entity-Component System** !

Diving into the making of my own game, I quickly realized that I was going to need a better structure than just using classes and inheritance to create and manage all of the game objects that I needed. That is when I stumbled upon the *ECS architecture*. 
Knowing that multiple examples of these already exist, I could have chosen to use them, but I decided not to. <br>
<br>
I wanted to learn.<br>
<br>
So here I went, learning about this architecture, and data-oriented design, and sparse sets, and a bunch of other things that comme into the making of a project like this.<br>
Throughout the making of this project, I heavily inspired myself from `EnTT`, another ECS that may be the best out there (so check it out if you really need great performance, Heim has still a lot to learn from it :) ).
